Cage of Movies

Users can:

  • Look up info about Nicolas Cage movies
  • Create an account and profile; deregistration is also possible
  • Add personal info and create a list of their favorite Nicolas Cage movies by adding or removing selections
  • Access the app and their profile from any device

Endpoints

Business Logic Endpoint URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all the movies
Return data (description, genre, director, image URL, year) about a single movie by title to the user /movies/:title GET :title None A JSON object holding data about the selected movie, structured like: { title: 'National Treasure' genre: ['action', 'adventure', 'mystery', 'thriller'] director: 'Jon Turteltaub' image URL: 'placholder.com/200' year: '2004' }
Return data about a genre by name /movies/genre/:genre GET :genre None A JSON object holding data about the selected movie's genres, structured like: { title: 'National Treasue', director: 'Jon Turteltaub', genres: 'Action, Adventure, Mystery, Thriller' year: '2004' } { title: 'The Weather Man', director: 'Gore Verbinski', genres: 'Comedy, Drama', year: 2005 }
Return data about a director (bio, brith year, death year) by name /movies/director/:director GET :director None A JSON object holding data about the selected director, structured like: { name: 'Jon Turteltaub' bio: 'Here are some facts about Jon Turteltaub.' birth year: '1963' death year: 'NA' }
Allow users to register /users/:newUser POST :newUser A JSON object holding data about the user, structured like: { username: "BenGates" email: "bgates@gmail.com" password: "Trea$ureHunter64" Date of Birth: 01/07/1964 } A text string: { "You've successfully registered, welcome!" }
Allow users to update their info /users/:username PUT :username None A text string: { "You've successfully updated your information!" }
Allow users to add a movie to their list of favorites, showing text that a movie has been added users/favorites/:title POST :title None A text string: { "You've added a new movie to your favorites!" }
Allow users to remove a movie from their list of favorites, showing a text that a movie has been removed users/favorites/:title DELETE :title None A text string: { "You've successfully deleted a movie from your favorites" }
Allow existing users to deregister, showing a text that a user email has been removed /users/:username DELETE :username None A text string: { "User account deleted, sorry to see you go!" }
