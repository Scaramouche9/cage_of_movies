Cage of Movies

Users can:

  • Look up info about Nicolas Cage movies
  • Create an account and profile; deregistration is also possible
  • Add personal info and create a list of their favorite Nicolas Cage movies by adding or removing selections
  • Access the app and their profile from any device

Endpoints

Business Logic Endpoint URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about all the movies
Return data (title, director info, genre, year) about a single movie by title to the user /movies/:title GET :title None A JSON object holding data about the selected movie, structured like: { title: 'National Treasure' director: { name: 'Jon Turteltaub', bio: 'bio', birth: '1920', death: 'NA' }, genres: 'Action, Adventure, Mystery, Thriller', year: '2004' }
Return data about a genre by name /genres/:genre GET :genre None A JSON object holding data about movies in the selected genre, structured like: { title: 'National Treasue', director: { name: 'Jon Turteltaub', bio: 'bio', birth: '1776', death: 'NA' }, genres: 'Action, Adventure, Mystery, Thriller' year: '2004' }
Return data about movies by the selected director (bio, brith year, death year) by name /directors/:directorName GET :directorName None A JSON object holding data about the selected director, structured like: { title: 'National Treasue', director: { name: 'Jon Turteltaub', bio: 'bio', birth: '1776', death: 'NA' }, genres: 'Action, Adventure, Mystery, Thriller' year: '2004' }
Allow users to register /users/:newUser POST :newUser A JSON object holding data about the user, structured like: { username: "BenGates" email: "bgates@gmail.com" password: "Trea$ureHunter64" Date of Birth: 01/07/1964 } A text string: { "You've successfully registered, welcome!" }
Allow users to update their info /users/:username PUT :username None A text string: { "You've successfully updated your information!" }
Allow users to add a movie to their list of favorites, showing text that a movie has been added /favorite/:newFavorite POST :newFavorite None A text string: { "You've added a new movie to your favorites!" }
Allow users to remove a movie from their list of favorites, showing a text that a movie has been removed /favorite/:deleteFavorite DELETE :deleteFavorite None A text string: { "You've successfully deleted a movie from your favorites" }
Allow existing users to deregister, showing a text that a user email has been removed /users/:username DELETE :username None A text string: { "User account deleted, sorry to see you go!" }
